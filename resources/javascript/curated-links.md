# JavaScript resources

Curated resources for developers learning JavaScript and looking to contribute to JavaScript open source projects.

## How to add a resource

1. Pick the right section (or propose a new one in your PR).
2. Add your entry in **alphabetical order** within the section.
3. Use this exact format: `- [Title](url) — one-line description.`
4. Keep descriptions to one sentence. If you can't, the resource probably belongs in its own section.
5. Link to the original source, not a mirror or aggregator.

Open a PR following [CONTRIBUTING.md](../../CONTRIBUTING.md). Branch name: `add-resource/your-name`.

## Learning the language

- [JavaScript.info](https://javascript.info/) — Modern, deep tutorial covering the language end to end.
- [MDN JavaScript Guide](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide) — The reference for what JavaScript actually does in browsers.

## Beginner-friendly project lists

- [First Contributions](https://github.com/firstcontributions/first-contributions) — A repo whose entire purpose is letting you make your first PR.

## Tools worth knowing

- [Node.js](https://nodejs.org/) — The JavaScript runtime used outside the browser.

## Newsletters and blogs

- [JavaScript Weekly](https://javascriptweekly.com/) — Curated weekly newsletter of JS news and tutorials.
