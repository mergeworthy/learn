# TypeScript resources

Curated resources for developers learning TypeScript and looking to contribute to TypeScript open source projects.

## How to add a resource

1. Pick the right section (or propose a new one in your PR).
2. Add your entry in **alphabetical order** within the section.
3. Use this exact format: `- [Title](url) — one-line description.`
4. Keep descriptions to one sentence. If you can't, the resource probably belongs in its own section.
5. Link to the original source, not a mirror or aggregator.

Open a PR following [CONTRIBUTING.md](../../CONTRIBUTING.md). Branch name: `add-resource/your-name`.

## Learning the language

- [TypeScript Handbook](https://www.typescriptlang.org/docs/handbook/intro.html) — The official guide, the best starting point for anyone coming from JavaScript.

## Beginner-friendly project lists

- [Type Challenges](https://github.com/type-challenges/type-challenges) — A community of TypeScript type system puzzles, useful for getting your hands dirty.

## Tools worth knowing

- [tsc](https://www.typescriptlang.org/docs/handbook/compiler-options.html) — The TypeScript compiler; understanding its config (`tsconfig.json`) is half the battle.

## Newsletters and blogs

- [TypeScript Weekly](https://typescript-weekly.com/) — Weekly newsletter focused on TypeScript news and articles.
